# Golden frame fixture — Pythagorean theorem, scene 003

This directory holds:

| File | Status |
|---|---|
| `scene_003.py` | hand-curated Manim scene (in repo) |
| `expected.png` | **missing** — generated by the first live render |
| `expected.meta.json` | optional metadata (frame index, resolution) |

`scene_003.py` is intentionally not LLM-generated so the golden frame
remains stable across model rolls. Total predicted runtime ~5.0s; the
test extracts the frame at t=2.0s (mid-rewrite between two MathTex forms).

## How `expected.png` is created

The first time someone runs `pytest tests/test_golden.py --update-golden`,
the test renders this scene, grabs the t=2.0s frame, and writes
`expected.png`. Subsequent runs compare against it with `PIL.ImageChops`,
allowing a per-pixel tolerance of 8/255 and a total diff of < 2%.

Until `expected.png` exists, `tests/test_golden.py` skips with a clear
xfail message. This lets the rest of the suite keep passing while the
golden fixture is bootstrapped on real hardware (CI / a maintainer's
box with MiKTeX and Manim CE installed).
